Deskripsi Chatbot

Chatbot ini dirancang untuk mempermudah pengiriman pesan secara otomatis melalui API Fonntee. Pengguna dapat memasukkan nomor telepon penerima serta pesan yang ingin dikirimkan. Chatbot juga dilengkapi dengan fitur untuk menentukan tanggal dan waktu pengiriman, sehingga memungkinkan pengguna untuk menjadwalkan pesan sesuai kebutuhan.

Fitur Utama

  1. Pengiriman Pesan Langsung atau Terjadwal
  2. Penggunaan API Fonntee
  3. Antarmuka yang Mudah Digunakan

Implementasi

  1. Pengiriman Pesan Promo untuk Pelanggan Secara Otomatis
  2. Pengiriman Pesan Ucapan Selamat Ulang Tahun Secara Terjadwal
  3. Pengingat Janji Temu atau Reservasi
  4. Follow-Up Layanan Pelanggan
  5. Pengiriman Notifikasi Pembayaran atau Tagihan
